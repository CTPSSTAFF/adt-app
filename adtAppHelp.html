CTPS ADT Data Browser Help

The tools/controls of the viewer are located above the map. These controls are used for navigating around the map and for performing queries of ADT traffic count data.

Tool Symbol Description

Pan/Zoom Tools
Pan At any time, you can pan the view by clicking and dragging the map with the four-arrow icon in any direction.
Zoom in Zooms in on the position you click or the box you define on the map.
Zoom out Zooms out from the position you click or the area you define on the map.
Go to previous extent Zooms back to the previous extent you were viewing. Use this option to go back to where you were before you zoomed or panned.
Go to next extent Zooms back to the next extent you were viewing. Use this option to un-do a Go To Previous Extent operation.
Go to initial extent Zooms to the initial extent of the map when the application was launched.

Show Information Tools
Query data by drawing a box Display information on specific traffic count locations on the map by dragging a box around a group of locations with the mouse. The information appears in table form in the bottom of the browser window.
Query data by submitting a form Find the location on the map of traffic counts meeting specified criteria. A search dialog window opens, with controls for choosing the year, town, and route number of traffic counts to be located. After clicking the "Submit Query" button in the search window, locations matching the riteria will be highlighted in yellow on the map.
Clear selection Clear Selection any highlighting shown around traffic count locations.

Other Tools
Download data Download either: (1) a compressed file in ZIP format containing the traffic count locations visible in the map, along with their associated count data. The locations are provided in shape file format (designed by Environmental Systems Research Institute) for use in a wide variety of geographic information systems software. Tabular data are provided in dBase IV format. or (2) a CSV file containing only the traffic count data.
